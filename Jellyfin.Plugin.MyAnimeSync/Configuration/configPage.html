Should be used with caution!
This option will determine if api requests are throttled for MyAnimeList.net
By disabling this, all requests will be sent to the api simultaneously, if myanimelist block an api request, the entry will be added as a failed update.
Your error logs may be flooded with failed updates.
If you are opting to disable this and use the "Sync Libraries" button, I highly reccomend you to manually force update every ~5 min to retry the failed entries.
This can be done by going in the dashboard -> Scheduled Tasks and executing the "MyAnimeSync retry to update failed entries" task.

Checked Libraries:

This will synchronize MyAnimeList.net with your jellyfin watch status for all checked libraries.
Use this with caution, make sure to backup your anime list on MyAnimeList.net before using this button.
Depending on the size of your libraries this can take while!
